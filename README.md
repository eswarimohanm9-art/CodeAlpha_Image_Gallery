# CodeAlpha Task 1 - Image Gallery

## 📌 Project Description
This project is a responsive Image Gallery developed using HTML, CSS, and JavaScript. Users can view images in a modern gallery layout, filter images by category, and open images in a lightbox view.

## 🚀 Features
- Responsive Image Gallery
- Category Filters (All, Nature, City, Animals)
- Image Hover Effects
- Lightbox Image Preview
- Mobile-Friendly Design
- Clean User Interface

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript

## 📷 Categories
- Nature
- City
- Animals

## 📁 Project Structure

CodeAlpha_Task1_ImageGallery/
│
├── index.html
└── README.md

## ▶️ How to Run
1. Download the project files.
2. Open `index.html` in any web browser.
3. Explore the gallery and filter images by category.
